## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.

## Tools Used
## Core Architecture
- TypeScript - A superset of JavaScript that adds static typing to catch errors early.

- Next.js - The React framework for production, handling routing and page structure.

- React - The library for building the modular user interface components.

- Zustand - A lightweight state management solution to keep the UI and math synced.

- Graphics & Physics
Three.js - The underlying 3D engine for rendering complex scenes.

- React Three Fiber - A React renderer for Three.js.

- Drei - A helper library for common 3D abstractions like cameras and lighting.

- Rapier - A fast 2D and 3D physics engine for realistic unfolding animations.

- Recharts - A charting library used to create the interactive 2D coordinate grid.

- Styling & Math
Tailwind CSS - A utility-first CSS framework for the app's minimal dark aesthetic.

- KaTeX - A fast math typesetting library for rendering matrices and formulas.

- Lucide React - A beautiful and consistent icon toolkit.

- Deployment & AI
Vercel - The platform used for hosting and automated deployment.

- Claude Code - An autonomous coding agent used to plan, write, and debug the majority of this project's codebase.
